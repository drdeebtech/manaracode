# Contributing

Thanks for your interest — but this repository is **not** open for contributions.

`manaracode` is published publicly as a **read-only portfolio**, so you can see
how the site is built. It is **not** open source. Issues, pull requests, and the
wiki are disabled, and the code is licensed for viewing only (see
[`LICENSE`](./LICENSE)).

You are welcome to **read** the code. You may not copy, modify, reuse, or
redistribute it.

If you'd like to work with us, report a security concern, or request permission
for any use beyond reading, please get in touch instead of opening a PR or issue:

- **Email:** contact@manaracode.com
- **Web:** https://manaracode.com
